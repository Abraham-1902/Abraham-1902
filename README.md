## 🚀 Proyecto Destacado: SauceDemo Automation

Framework profesional desarrollado para validar el flujo completo de autenticación y seguridad.

* **Arquitectura:** Implementación de **Page Object Model (POM)** para una clara separación entre la lógica de negocio y los scripts de prueba.
* **Estrategia de Datos:** Uso de **Data-Driven Testing** mediante archivos **JSON**, permitiendo probar múltiples perfiles de usuario sin duplicar código.
* **Documentación:** Incluye un **Plan de Pruebas** formal con Matriz de Trazabilidad para asegurar la cobertura del Sprint.
* **Reportabilidad:** Generación automática de reportes visuales en **HTML** para análisis de resultados.

🔗 [Ver Repositorio del Proyecto](https://github.com/Abraham-1902/playwright-automation-tests)
